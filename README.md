# vue

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
### Change default baseUrl in /src/main.js
```
axios.defaults.baseURL = 'http://apiroute';
-- change http api route to your local api url
```
### Compiles and minifies for production
```
npm run build
```
